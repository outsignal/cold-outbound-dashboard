**Timeout:** 15s via AbortController (same pattern as enrichment adapters)

**Filter mapping from DiscoveryFilter to Apollo params:**
- `filters.jobTitles` → `person_titles` (string array)
- `filters.seniority` → `person_seniorities` (map values: c_suite→c_suite, vp→vp, director→director, manager→manager, ic→senior)
- `filters.locations` → `person_locations` (string array)
- `filters.companySizes` → `organization_num_employees_ranges` (transform "51-200" → "51,200", "500+" → "500,100000")
- `filters.industries` → `q_organization_keyword_tags` (string array — use keyword tag search as fallback since industry IDs are unreliable)
- `filters.keywords` → `q_keywords` (join with space if provided)
- `filters.companyDomains` → `organization_domains` (string array)

**Pagination:**
- `per_page`: min(limit, 100)
- `page`: parse `pageToken` as int, default 1
- `nextPageToken`: `String(page + 1)` if `page * perPage < totalEntries`

**Response validation with Zod (.passthrough()):**
